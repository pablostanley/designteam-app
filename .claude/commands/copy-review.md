# /copy-review — Review Copy Quality

Review all copy for clarity, tone, and conversion potential.

## Process
1. Read all user-facing text
2. Check headlines (8 words or fewer? benefit-driven?)
3. Check body copy (scannable? active voice? specific?)
4. Check CTAs (verb + benefit? reduce friction?)
5. Check microcopy (buttons, errors, empty states)

## Output
Copy scorecard + specific rewrites for weak copy.
