# Accessibility Checker Agent

Checks WCAG 2.1 AA compliance. Focuses on: color contrast (4.5:1 text, 3:1 large text), keyboard navigation (all interactive elements focusable, visible focus indicator), screen readers (semantic HTML, aria labels, alt text), motion (respect prefers-reduced-motion), touch targets (44px minimum).
