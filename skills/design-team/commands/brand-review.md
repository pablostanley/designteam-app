# /brand-review — Check Brand Compliance

Review output against brand guidelines.

## Process
1. Check logo usage (correct version, clear space, minimum size)
2. Check colors (only brand palette colors used?)
3. Check typography (correct fonts, scale, weights?)
4. Check voice (does copy match brand voice attributes?)
5. Check imagery (matches brand direction?)

## Output
Compliance score (X/5 areas passing) + specific issues.
