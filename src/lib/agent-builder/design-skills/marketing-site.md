# Marketing / Product Website Design

You're designing a multi-section marketing site — storytelling through scroll.

## Core Principles

- **Each section = one argument.** Build a case for the product. Section by section, the visitor should become more convinced.
- **Progressive disclosure.** Start with the big picture (hero), then get more specific (features, how it works, pricing).
- **Visual rhythm.** Alternate between full-width and contained, text-heavy and visual, light and dark.
- **Momentum toward CTA.** Every section should build desire. The final CTA should feel inevitable.

## Structure

1. **Nav**: Clean, minimal. Logo + key links + CTA button.
2. **Hero**: Big promise. Not features — outcomes. "Ship faster" not "CI/CD pipeline."
3. **Social proof bar**: Logo cloud or key metrics. Builds immediate trust.
4. **Features**: 2-3 sections, each highlighting a key capability. Mix layouts.
5. **How it works**: 3-step process or visual walkthrough.
6. **Testimonials**: Real people, real results. Quote + name + company + avatar.
7. **Pricing** (optional): Clear tiers. Highlight recommended plan.
8. **Final CTA**: Different framing from hero. "Join 10,000 teams" or "Start free."
9. **Footer**: Links, legal, social.

## Section Variety

CRITICAL: No two adjacent sections should have the same layout. Alternate between:

- Centered text -> Left-aligned text + right image
- Card grid -> Full-width statement
- Light background -> Dark background
- Spacious (py-24) -> Tighter (py-16)
