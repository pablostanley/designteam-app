# Presentation / Slide Deck Design

You're designing slides — NOT a website. Completely different rules apply.

## Core Principles

- **ONE idea per slide.** If you need two bullet points, that's two slides.
- **HUGE typography.** text-6xl to text-9xl for main points. If it's not readable from across the room, it's too small.
- **Minimal content.** Slides support the speaker, they don't replace them. Max 20 words per slide.
- **Strong visual rhythm.** Alternate between text slides, image slides, and accent slides.

## Slide Types

- **Title slide**: Product name + tagline. Bold, centered, dramatic. Full-bleed color or image.
- **Statement slide**: One sentence, huge type, centered. The most powerful slide type.
- **Data slide**: One metric, enormous (text-8xl or text-9xl). Small context text below.
- **Image slide**: Full-bleed image with overlay text OR image left + text right.
- **List slide**: 3-4 short items MAX. Use clear visual hierarchy for each item.
- **Section divider**: Full-bleed accent color + section title. Marks transitions.
- **CTA/closing slide**: Clear next step. Contact info or URL.

## Artboard Setup

- Use 1920x1080 (16:9) or 1280x720 artboards for standard presentations.
- Position side-by-side: each slide offset by width + 100px.
- Name slides: "01 — Title", "02 — Problem", "03 — Solution", etc.

## Color Strategy

- Pick ONE bold accent color. Use it for emphasis slides and key data.
- Dark backgrounds (zinc-950, slate-900) make text pop — prefer dark for presentations.
- Alternate between dark bg + light text and accent bg + white text for rhythm.
- NEVER use more than 3 colors total.
