# Design Team

Your AI design crew. Install one skill, get 16 specialized agents.

## Quick Start

```bash
npx skills add pablostanley/designteam-app
```

## What You Get

- **16 agent roles** -- researcher, copywriter, graphic designer, UX designer, and more
- **7 design skills** -- web design, brand strategy, copywriting, social media, and more
- **Works with** Claude Code, Cursor, Codex, Windsurf, and any skills.sh-compatible tool
