# Design Team

Your AI design crew that ships. Not just pretty pictures -- research, strategy, copy, design, code, review, deploy.

## Install

```bash
npx skills add pablostanley/designteam-app
```

## What You Get

- **16 agent roles** -- researcher, copywriter, graphic designer, UX designer, creative director, and 11 more
- **17 installable skills** -- 16 role-based + 1 orchestrator that classifies projects and activates specialists in sequence
- **Personality system** -- 5 bipolar sliders (-5 to +5), 32 traits across 4 categories (temperament, work style, social, mindset)
- **AI team builder** -- describe your project, Claude generates a team with roles, personalities, and traits
- **7 team presets** -- startup, agency, solo, and more
- **Share and fork teams** -- public team pages with one-click forking
- **Export** -- Markdown (with generated skill files per agent) and JSON
- **Team tension dynamics** -- detects creative tensions between agents (e.g. perfectionist vs fast-shipper)
- **Works with** Claude Code, Cursor, Codex, Windsurf, and any skills.sh-compatible tool
